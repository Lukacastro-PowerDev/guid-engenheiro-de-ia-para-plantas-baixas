🏗️ Chatbot Inteligente para Planejamento de Obras
IA Generativa + Análise de Planta Baixa + RAG
📌 Visão Geral
Este projeto apresenta o desenvolvimento de um chatbot inteligente baseado em IA Generativa, capaz de analisar plantas baixas, fornecer insights técnicos, sugerir melhorias de projeto e auxiliar no planejamento e controle de obras.
A solução utiliza modelos multimodais (texto + imagem) e é aprimorada com RAG (Retrieval‑Augmented Generation), permitindo que as respostas sejam fundamentadas em conhecimento técnico especializado da construção civil.

🎯 Objetivo do Projeto
Desenvolver um chatbot funcional que:

Interaja em linguagem natural
Analise plantas baixas enviadas pelo usuário
Gere insights técnicos e sugestões de melhoria
Apoie o planejamento de obras
Auxilie no controle e previsão de materiais
Recomende boas práticas da construção civil
Demonstre uso real e aplicado de IA Generativa com RAG


🧠 Tecnologias Utilizadas

Google AI Studio (Gemini API)
Modelo Multimodal (Texto + Imagem)
IA Generativa
RAG (Retrieval‑Augmented Generation)
Base de conhecimento técnico (normas, guias e boas práticas)


🗂️ Arquitetura da Solução

O usuário envia:

Uma planta baixa (imagem ou PDF)
Uma pergunta ou solicitação em linguagem natural


O chatbot:

Interpreta visualmente a planta
Consulta a base de conhecimento (RAG)
Gera uma resposta contextualizada com IA Generativa


O usuário recebe:

Análise técnica do projeto
Sugestões de melhoria
Recomendações de planejamento e execução




🔎 Uso de RAG (Retrieval‑Augmented Generation)
O chatbot utiliza RAG de forma prioritária, acessando uma base de conhecimento técnico que pode conter:

Guias de planejamento de obras
Boas práticas construtivas
Checklists de execução
Recomendações de segurança
Procedimentos de organização do canteiro
Diretrizes de sustentabilidade

Benefícios do RAG:

Redução de respostas genéricas
Maior precisão técnica
Melhor alinhamento com práticas reais de obra
Evita alucinações do modelo

Sempre que o conteúdo estiver disponível na base RAG, ele terá prioridade sobre o conhecimento geral do modelo.
